# Cara deploy ke GitHub Pages

1. Isi folder `songs/` dengan MP3 (lihat `songs/README.txt`). Boleh dilewati dulu — web tetap jalan tanpa musik.
2. Buat repo baru di GitHub (saran: nama acak biar gak ketebak, mis. `fs-0208-rewind`), **Public**.
3. Upload isi folder `rewind/` ini (index.html + folder songs) ke root repo — bisa drag & drop di github.com.
4. Repo → **Settings → Pages** → Source: `Deploy from a branch` → Branch: `main` / root → Save.
5. Tunggu ±1 menit. Link jadi: `https://<username>.github.io/<nama-repo>/`
6. Tes dulu di HP kamu, baru kirim ke Firda tanggal 2 Agustus 🎉

## Cara pakai (buat Firda)
- Tap **MULAI** → musik jalan otomatis
- Tap kanan = lanjut, tap kiri = mundur, tahan = pause
- Slide quiz harus dijawab dulu baru bisa lanjut
- Paling enak dibuka di Chrome/Safari HP, portrait, pakai headset

## Catatan
- File `_chat.txt` JANGAN ikut di-upload ke repo ya (privasi).
- MP3 di repo publik bisa diunduh siapa saja yang tahu link — nama repo acak sudah cukup aman untuk ini.
